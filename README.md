# 🛠️ ScriptCat - Collection 

A minimal toolkit of custom JavaScript userscripts, web modifications, and browser tweaks.

## 📌 Overview

Instead of hunting down fragmented code snippets, this repository serves as a single source of truth for high-caliber, battle-tested web modifications.

## 🚀 How to Use

1. Install a userscript manager like **[ScriptCat (Recommended)](https://scriptcat.org/)** or **[Tampermonkey](https://www.tampermonkey.net/)**.
2. Navigate to the script you want to install.
3. Click the **Raw** button on GitHub, and your extension will prompt you to install it.
